## Stack input

0. `Integer`: numerator
1. `Integer`: denominator.

## Stack output

0. `Integer`: result of the integer division. If the denominator is 0, the result will be 0.

## Examples

With the top stack value as 10 and the second stack value as 10, the result is 1.
With the top stack value as 1 and the second stack value as 2, the result is 0.

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
