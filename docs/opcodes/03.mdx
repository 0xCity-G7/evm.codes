## Stack input

0. `Integer`: first value.
1. `Integer`: value to subtract to the first.

## Stack output

0. `Integer`: result of the subtraction modulo 2<sup>256</sup>.

## Examples

With the top stack value as 10 and the second stack value as 10, the result is 0.
With the top stack value as 0 and the second stack value as 1, the result is 0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF.

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
