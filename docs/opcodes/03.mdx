## Stack input

0. `Integer`: first integer.
1. `Integer`: integer to subtract to the first.

## Stack output

0. `Integer`: result of the subtraction modulo 2<sup>256</sup>.

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
