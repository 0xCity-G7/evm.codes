## Stack input

0. `Integer`: left side integer.
1. `Integer`: right side integer.

## Stack output

0. `Integer`: 1 if the left side is equal to the right side, 0 otherwise.

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
