## Stack input

0. `Integer`: first value to add.
1. `Integer`: second value to add.

## Stack output

0. `Integer`: result of the addition modulo 2<sup>256</sup>.

## Examples

With the top stack value as 10 and the second stack value as 10, the result is 20.
With the top stack value as 0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF and the second stack value as 1, the result is 0.

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
