## Stack input

0. `Integer`: first value to multiply.
1. `Integer`: second value to multiply.

## Stack output

0. `Integer`: result of the multiplication modulo 2<sup>256</sup>.

## Examples

With the top stack value as 10 and the second stack value as 10, the result is 100.
With the top stack value as 0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF and the second stack value as 2, the result is 0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFE.

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
